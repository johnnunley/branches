=== INSTRUCTIONS FOR BUILDING ===

Mac and Linux clients:

First, install CMake, make, and both gcc and g++, if you haven't already. While within this folder, run the commands:

$ cmake -G "Unix Makefiles"
$ make

Windows clients: (note: in beta testing)

First, install CMake. Then, run this command inside of the folder:

> cmake -G "Visual Studio 15 Win64"

=== INSTRUCTION FOR RUNNING ===

Type in the command:

> ./branches
